
### Create Propositional Logic

**Setup**

1. select index.html in frontend folder
then open terminal or powershell or cmd  and run this command in your shell
2.  cd backend 
3. npm install 
4. node .

-----------------

### تصمیم پذیری منطق گزاره ای
**نصب**

1. فایل index.html را در مرورگر باز میکنیم یا روی فایل کلیک کنید 
 بعد از باز شدن باید بک اند فایل را اجرا کنیم برای این کار cmd یا powershell را اجرا کنید (توجه در داخل پوشه اصلی فایل باید اجرا کنید)

2. cd backend
3. npm install 
4. node .

فایل اجرا خواهد شد
